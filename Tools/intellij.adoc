= Intellij keyboard shortcuts

* ⇧ Shfit
* ⌃ Ctrl
* ⌥ Opt/Alt
* ⌘ Cmd/Super

== Essentials

|===
|Command | What

| ⌥ Enter | Open context menu
| ⇧ ⇧ |  Search everything
| ⌘ ⇧ A |  Find actions
| ⌘ O | Find Class
| ⌘ ⇧ O |  Find Files
| ⌘ ⌥ O |  Find Symbols
|===

|===
| Command | Vim | What

| ⌃ Space | | Complete
| ⌃ Space ⌃ Space | | Futher completion
| ⌃ Up/Down | | Code selection
| ⌘ / | | Comment line
| ⌥ ⇧ Up/Down | | Move line
| ⌘ ⇧ Up/Down | | Move function

| ⌘ - | zc | Fold Function
| ⌘ = | zo |Unfold function
| ⌘ ⇧ - | | Fold all
| ⌘ ⇧ = | | Unfold all

| ⌘ ⌥ T | | Wrap line
| ⇧ F6 | | Rename
| ⌘ ⌥ V | | Extract variable
| ⌘ ⌥ M | | Extract method
|===
